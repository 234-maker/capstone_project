# Capstone Project

A full-stack capstone project for the **AI-assisted development track**, built with a React frontend and a Node.js backend.

## Tech Stack

- **Frontend:** React (in `client/`)
- **Backend:** Node.js (in `server/`)
- **Tooling:** npm, Git

The repo is organized as a monorepo with separate `client/` and `server/` packages. See [CLAUDE.md](./CLAUDE.md) for folder structure and development conventions.
